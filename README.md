# Планировщик задач

Простое веб-приложение для планирования и управления задачами с простым интерфейсом

# Описание

- Планировщик задач предствляет собой веб-сервер на Go с фронтендом для управления задачами, в функции управления входят: создание, изменение, удаление и отметка задачи как выполненной.

# Возможности:
- Создание, изменение и удаление задач
- Отметка задачи как выполненной
- Возможность ввода пароля в качестве защиты от несанкционированного доступа  

# Задания со звездочкой:

- Выполнено всё, кроме задания с неделями и месяцами

# Инструкция по запуску кода локально:
- В файле .env укажите желаемый порт
- Создайте образ make run
- Запустите контейнер make 

# Тесты
- В файле tests/settings.go укажите желаемые параметры
- Запуск тестов: make test

# Docker

- Сборка образа: make docker-build
- Запуск контейнера: make docker-run

- Приложение будет доступно по В браузере по адресу http://localhost:7540
